# CryptoXchange
CryptoXchange is a Java-based crypto exchange simulator showcasing OOP, data structures, and backend system design. It features wallet management, spot trading, market price simulation, modular services, and scalable architecture, built for learning real-world trading systems.

A console-based cryptocurrency exchange simulator built using Core Java and OOP principles.

## Features
- User authentication (simulated)
- Wallet & balance management
- Spot trading (Buy/Sell)
- Market price simulation
- P2P & gamified trading (extensible)
- Modular architecture

## Tech Stack
- Java
- OOP Principles
- Collections Framework
- Randomized Market Simulation

## Project Structure
- model → Core entities (User, Wallet, Trade)
- service → Business logic
- util → Helpers and simulators
- app → Entry point

## Learning Outcomes
- Object-Oriented Design
- Java Collections
- System design for exchanges
- Trading simulation logic

## Future Scope
- Order matching engine
